1. Składnia teoretyczna

<link rel="stylesheet" href="style.css">

2. Gdzie wstawiany jest styl zewnętrzny, jaki znacznik jest za to odpowiedzialny oraz ile razy może on występować?

Styl zewnętrzny jest wstawiany w sekcji <head> dokumentu HTML. Znacznikiem odpowiedzialnym za podłączenie arkusza stylów jest <link>. Może on występować wielokrotnie, aby podłączyć różne arkusze stylów.

3. Najprostsza forma wstawiania stylu zewnętrznego

<link rel="stylesheet" href="style.css">

4. Ile arkuszy stylów może być dołączonych oraz jaka zasada musi być zachowana?

Można dołączyć wiele arkuszy stylów do jednego dokumentu HTML. Ważne jest, aby kolejność ich wczytywania była odpowiednia, gdyż style później załadowane mogą nadpisywać wcześniejsze.

5. Co się dzieje w przypadku konfliktów?

W przypadku konfliktów (czyli gdy te same style są definiowane w różnych arkuszach CSS), decyduje ostatnia załadowana definicja lub selektor o wyższej specyficzności.

6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

Pliki arkuszy stylów mogą być tworzone i edytowane w dowolnym edytorze tekstowym, np. Notatniku, zapisując plik z rozszerzeniem .css.

7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

Specjalne edytory CSS, np. Visual Studio Code, Sublime Text, lub edytory online, pozwalają na wygodne pisanie i podgląd kodu CSS.

8. Jakie rozszerzenie posiada plik będący zewnętrznym arkuszem stylów?

Plik zewnętrznego arkusza stylów posiada rozszerzenie .css.

9. Jakie są zalety zewnętrznego arkusza stylów?

  • Ułatwia zarządzanie stylem w wielu dokumentach HTML.
  • Zmniejsza rozmiar pliku HTML, czyniąc go bardziej czytelnym.

10. Uwagi na temat pisania w CSS

Podczas pisania CSS należy unikać nadmiernego używania !important, dbać o czytelność kodu oraz wykorzystywać selektory w sposób efektywny.
