Pojęcie Co znaczy skrót ID? Jaki jest cel stosowania ID? Jakie jest znaczenie znaku #? Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Uwagi na temat stosowania ID (trzy konieczne wyliczenia)
ID Identifier – identyfikator Służy do jednoznacznego określenia jednego elementu na stronie Symbol # służy do odwoływania się do ID w CSS lub JavaScript Przykład użycia:
<div id="blok1">...</div>
Wywołanie w CSS:
#blok1 { color: red; }
Znaczenie: styl jest zastosowany tylko do elementu z tym ID
  1. Każde ID na stronie musi być unikalne.
  2. ID nadaje się idealnie do manipulacji za pomocą JavaScript.
  3. Używaj ID tylko wtedy, gdy naprawdę musisz wskazać konkretny element.
